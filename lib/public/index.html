Bilbo's Books

Local Book Store

Route Method Description Example 
/books
 GET Gets a list of all books at the book store List Books 
/books/1
 GET Returns a single book at the store Get Book 
/books
 POST Adds a new book and author 
                curl -d '{ title: 'bookie', publisher: 'dunno', released: 1960, authorIds: [1, 2], newAuthor: { dob: '01/01/2022', pob: 'Bend, OR', name: 'Testing New Author' }}' -H 'Content-Type: application/json' 'https://alchemy-book-shoppe.herokuapp.com/books'
 
/authors
 GET Gets a list of all authors at the book store List Authors 
/authors/1
 GET Returns a single author and all of their books at the store Get Author 
/authors
 POST Adds a new author with corresponding books 
                curl -d '{ dob: '01/01/2021', pob: 'Portland, OR', name: 'Some Author', bookIds: [1, 2] }' -H 'Content-Type: application/json' 'https://alchemy-book-shoppe.herokuapp.com/books'
